# /compact-guide 커맨드

## 사용법
```
/compact-guide               # 현재 상태 및 가이드
```

## 컨텍스트 관리 원칙
```
"컨텍스트는 신선한 우유. 시간이 지나면 상한다."
```

## 토큰 상태 표시
```
📊 컨텍스트 상태

현재: 72,000 / 100,000 토큰 (72%)

[██████████████░░░░░░] 72%

상태: ⚠️ 정리 권장
```

## 단계별 대응

### 50% 미만 (안전)
```
✅ 정상 작업 진행
```

### 50-70% (주의)
```
⚠️ 권장 조치:
- 불필요한 파일 읽기 최소화
- 큰 파일은 필요한 부분만 읽기
- /checkpoint 주기적 실행
```

### 70-85% (경고)
```
🟡 권장 조치:
- /checkpoint 실행
- 복잡한 새 작업 시작 자제
- 현재 작업 마무리 집중
```

### 85% 이상 (위험)
```
🔴 필수 조치:
1. /handoff 즉시 실행
2. 현재 작업 상태 저장
3. 새 세션 시작 권장
```

## 토큰 절약 팁
```
1. 파일 읽기
   ❌ 전체 파일 읽기
   ✅ 필요한 라인만 읽기 (offset, limit)

2. 검색
   ❌ 넓은 범위 검색
   ✅ 구체적 패턴 + 경로 제한

3. 출력
   ❌ 긴 설명
   ✅ 핵심만 간결하게

4. 반복
   ❌ 같은 파일 여러 번 읽기
   ✅ 한 번 읽고 기억 활용
```

## 압축 전 자동 저장
```
컨텍스트 압축 시:
1. 현재 작업 상태 자동 저장
2. HANDOFF.md 생성 제안
3. 중요 컨텍스트 유지
```

## 관련 커맨드
- `/handoff` - 세션 인계
- `/checkpoint` - 상태 저장
